整理思路

现在是七月中旬了，嘴上说着不急是假的。Java是大二的时候开始学的，中间兜兜转转学了C和C++，Python。同时又把计算机基础学了一遍。Java真正学了多久呢，我也不好计算。

最开始的时候学完基础语法感觉好复杂，再加上没有实际的应用。后续就转Python学习了，那个时候看着别人的疫情打卡，(毕业了应该管不到我了，当时觉得这玩意儿是真烦。)让每天的打卡都自动化。再到后来的刷各种学校安排的任务，什么四史学习，英语的welearn刷课。还有学习通的刷课，(这玩意儿是真的好用)。成功的用Python减少了大学生活中一些重复机械缺没有实际意义的东西。还有步道乐跑。

大二做到什么地步了呢，基本上布置个环境，写个简单的调用没啥问题。真正开始爆发式学习应该是到大三了。开始接触计算机的四件套，开始学数据结构，这中间把C和C++的基础给学了，就是那种勉强能用吧。但为之后的学习打下坚实的基础。至于Java。鬼知道扔哪去了。除了会se的基本概念，多线程都调用不利索。

然后就是考研，做毕设。这就不得不把事情都给暂停了。土木的毕设还是挺费时间的。三万多字，一百页的Word，十四张A2图纸，这确实是一辈子忘不掉。中间想过跟计算机结合到一起。结果最后也就做了一个Excel编程，也就是数据处理。当然要我现在去做三个月不仅仅能把毕设做完，同时还能把Excel表格给换成python或者Java程序。但可惜时间不等人啊。当时也是顺便用markdown把使用Excel和解释整个毕设的思路给记录下来，同时用git给同步到gitee上。算是给学校留下一点能见证的东西，就是没啥人用就是了。

然后就是到今年年初，开始找工作吧，这其中也发生不少事，因为最开始的时候用Python还算熟练，但是等做完Python的商业使用，却发现市场上没有Python的工作机会，那些工作基本上是基于数据处理，数据清洗，还有AI的调用。

当时怎么想的呢，因为数据清洗还需要重点学习number和panda这两个Python库，也就是相当于重头再来，而且实际意义上这玩意儿也只是做个数据处理，如果当做副业做还可以，但是全部的工作内容是这个我是有点无法接受。

我的目标是经过几年的沉淀，我可以用程序实现我所有想干的事，如果换个说法就是全栈开发，这种中间的数据处理不是我的目标。至于AI，那就是调用openai库。这个可行性很高，同时也有点基础，大学的时候痴迷做过公众号，所以在公众号的后台调用过自动回复，用自己的服务器和API接口。后来就是十五美元用完了，再加上腾讯自己出了一个智能对话。于是这个项目就不了了之了。

再加上因为在调用智能对话之前，其实也算是做过类似的吧，通过nonebot，做了一个QQ机器人，当时就了解了下所谓的回复内容什么之类的，再加上对AI的浅显认知，以为没啥两样。但最破防的就是，我在广州这边投递相关的，没有。那也确实要说调用个API是挺简单的，但是作为企业来说，肯定是自己去实现AI框架最为合适，但是这个框架内部要用到神经网络，从浅显的理解上就知道这玩意儿给本科生是做不出来的。挺复杂。所以Python找工作计划，破产。

接下来就是重点了，当时已经三月份了，发现Python的路都堵死了，只能再考虑下其他语言了，当时摆在面前的有两条路，一个是C++，一个是Java。Java岗位多，但是都很卷，要掌握的内容很多。C++则是市场环境没那么大，并且主要还是QT开发。

当时摆在面前的就这两条路，于是在经过一番思考后，选择学习Java。一个是市场规模大，虽然很卷，但是如果详细查看招聘页面的话，其实发现Java可以不用，但是不可以不会。所以这才是关键点。C++，对当时的我来说，各种观念还是太过深奥了，当时这俩都是半吊子，但是Java相对来说没有太复杂的概念，寻找资料还是比较方便。C++就不同了，虽然我详细的看完过C primer plus和部分的C++ primer plus。但是最大的感受就是，几百多页的书，撑死就把理论给讲清楚了。但是还不够。再加上市场规模没有那么大，选择Java。

于是经过四个月的学习，把se的内容过了四遍，ee的内容过了三遍，也算是Java入门了，顺便把网上比较火的开元项目给做了一个半。

ps(主要找的教程有点凌乱，廖雪峰大佬的教程永远就是第一遍看不懂，第二遍看个大概，第三遍才真正懂他的思想，我python的坑就是他给挖的。没点基础看了相当于没看)但不得不说廖雪峰大佬的思想真的很高，建立在一个很高的高度上去解释这件事。同时结合着Java源码解析，学习记录博客，磕磕绊绊的走过来了。

然后就开始了找工作的旅程，也就是在这里要有自我的思考了。

深度思考

主要是这段时间投简历没人回，就算是回了也是说对不起，不合适。

所以才开启了这进一步的思考。

环境

大环境确实不好，毕业已经一年，虽然没有进入过市场，但是我有室友和朋友进入职场了。首先是土木行业，从最开始的五千涨到一万，实习期转正。再到一万降薪百分之三十成为七千。从这一点表明大环境堪忧。本身工地都是属于全年无休，真正想去也是要考虑这一行业不能长期发展，过三五年之后要成家，但是全年无休以及居无定所，足以劝退很多人。但是这个行业也开始降薪就代表了市场上整体的趋势堪忧。

电力行业，我表哥干了两年的电力行业，依旧是三四千工资，就算工厂离职人数多，但是依旧不考虑加薪或者提升职位来保留剩余的人，那就说彻底烂完了。现在在家待业一个月了，还是没有工作。

律师，这玩意儿必须要有律师资格证，这玩意儿每年通过的人数没多少，基础薪资也只有两千，不过却活的挺自由。

计算机，当初学习Java的时候开过直播，有一些被裁员的人到直播间里吐槽过，看的确实是没话说。

从多个行业去说明，当前的趋势是各种工作都受到很大的影响。除了基础劳动力，这玩意儿是真缺人。

那么当前大环境下什么行业比较好找工作呢，工厂，缺人。直播，缺少剪辑，虽然也不缺，但是比其他还算好进。AI,缺人,但是一样,缺的是中高级,市场广袤,但是架不住市场紧缺的是中高端,我这里只是列举一些初级人员.毕竟我卡在这一步,所以别说前景很好，但是入门依旧很难。

自身优势

去找工作说白了就是把自己给卖出去，针对市场的初级人员岗位我有什么样的条件。

中高级不是自学能搞定的，需要实际经历。

技能 程度
Java 熟悉基本语法，熟悉spring ,spring MVC，spring Cloud。等框架
Java相关 maven，一个构建和版本管理工具，activeMQ Artemis，rabbitMQ，Kafka这仨JMS消息服务协议。swagger，一个open API，主要针对web应用的调试
数据库 MySQL，Redis。一个是默认存储，一个是提升性能可以做缓存。
Python 基础语法，一些基本库的调用，复杂的基础概念。
前端 前端三件套，vue还有node这些框架就是能看懂就完事。
算法 常见的算法知道，手写也行，就是有点费劲
环境和版本控制 基本的Linux操作，docker的使用，git版本控制，github代码存储仓库。虚拟机的使用。
计算机相关的乱七八糟的 在服务器上布置梯子，搭建个博客，计网，操作系统，组成原理，数据结构，是对当前的我够用
其他乱七八糟的 CAD，PKPM，等这些土木的东西还是会用的。PS，视频剪辑，简单的会用，多吊炸天的效果我也做不出来。

把真实的技能点列举出来，基本上满足当前大部分公司对于计算机从业者的要求，针对于招聘页面上写的东西。

下一步是什么，主要也就是简历的编写了，把这些东西怎么系统全面的展示出来是个问题，专门写了个博客来列举简历的编写技巧，不过没啥用。还是卡在第一步，没多少人愿意去看简历，或者简历看完就不回复。

那么就剩简历的优化了，确实很麻烦，但是不得不去立马处理的。头大。

项目

实际做过的Java项目很少，毕竟实话说一个项目的开发周期一两个月不是啥问题。就算是有现成的成品去照抄也是需要一段时间的沉淀。

但同样的，别人也不会了解你的项目。也是根据只言片语来提问，所以这就有了可以钻漏洞的。那就是看项目源码自己去思考会提问哪些。

那么就是去github上找开源的项目吧。
